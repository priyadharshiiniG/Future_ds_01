E-Commerce Insights Dashboard

This project is an interactive E-Commerce Analytics Dashboard built using Power BI, designed to help businesses analyze sales, profit, customer behavior, and product performance efficiently. The dashboard provides clear and actionable insights using data visualizations, KPIs, and drill-down analysis.
